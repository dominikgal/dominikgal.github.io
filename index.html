KODI custom Wizard

Install latest addons for watching TV Shows and Movies

WIzard --> v.0.0.3

Build --> v.0.0.5

  • Go to KODI file manager
  • Click add source
  • Path source is https://dominikgal.github.io/kodiWiz/ (Give it name ".kodiWiz")
  • Go to addons, click install from zip, click .kodiWiz, click plugin.program.KodiWiz
  • Wait for wizard install and follow instructions or go to program addons and install build from there
